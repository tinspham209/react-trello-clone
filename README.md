# React Drag'n'Drop

Learn to create a Trello Dashboard clone with React-DnD!
Link tutorial: https://youtu.be/aK2PD_REk7A

## Installation

Use the cmd line to install dependencies.

```
npm install
```

## Usage

```
npm run build
npm run dev
http://localhost:4000/
```
